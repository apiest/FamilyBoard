# Documentation assets

This folder holds README assets (screenshots, diagrams).

## screenshot.png

The main dashboard screenshot referenced from the top of `README.md`.

Suggested specs:

- 1280–1920 px wide, 16:9 or 16:10
- PNG, < 500 KB after optimization (use `pngquant` or `oxipng`)
- Capture the FamilyBoard dashboard with realistic-but-anonymized data
  (use the `Alice` / `Bob` / `Member 1` style from the dev container)

Replace this folder's `screenshot.png` whenever the UI changes.
